# Distribuicao.DadosAgrupados

Esse projeto foi desenvolvido para fixação do conteúdo da disciplina de Estatistica do curso Tecnólogo de Analise e Desenvolvimento de Sistemas. Ele serve de introdução para o desenvolvimento de uma platforma de classificação de dados para uso com algoritimos de aprendizado de máquina.

Uso: Atualmente só funciona com agrupamento de dados numéricos discretos ou contínuos.

Argumentos: 1. Valores numeros discretos ou continuos separados por virgula.
